Blockquotes

This is a blockquote. It will be displayed in a block. It can be used to quote someone.
Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.

Blockquotes can be formatted with the '<blockquote>' and "</blockquote>" tags.

Text Formatting

Strong Text can be formatted with the '<strong>' and '</strong>' tags. Use Strong Text when you want an element to have extra importance

Else, just use Bold Text with the '<b>' and '</b>' tags if you want to highlight some text without bringing extra importance to it.


Emphasized Text can be formatted with the '<em>' and '</em>' tags. Use Emphasized Text when you want an element to have importance

Else, just use Italic Text with the '<i>' and '</i>' tags if you want to highlight some text without bringing extra importance to it.


Marked Text can be formatted with the '<mark>' and '</mark>' tags.


Deleted Text can be formatted with the '<del>' and '</del>' tags.


Inserted Text can be formatted with the '<ins>' and '</ins>' tags.


Subscript Text can be formatted with the '<sub>' and '</sub>' tags.


Superscript Text can be formatted with the '<sup>' and '</sup>' tags.
